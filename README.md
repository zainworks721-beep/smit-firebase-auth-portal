# 🔐 SMIT Firebase Authentication Portal

A secure and modern authentication system built using **Firebase Authentication** with a clean SMIT student portal UI.  
This project includes login, signup, form validation, and dashboard redirection with toast notifications.

---

## 🚀 Features

- 🔑 User Signup with Email & Password
- 🔓 User Login System
- 📅 DOB Validation (age & format checks)
- 📧 Email & Password Regex Validation
- ⚠️ Real-time Error Handling
- 🔔 Toast Notifications (Success / Error)
- 🔄 Auto Redirect after Login (Dashboard)
- 👁️ Dynamic UI (Login / Signup Toggle)
- 🔥 Firebase Authentication Integration

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase Authentication

---


## ⚙️ How It Works

1. User enters email & password
2. Input is validated using regex + custom rules
3. Firebase handles authentication
4. Success → Redirect to dashboard
5. Errors → Displayed via toast notifications

---

## 🔥 Firebase Features Used

- createUserWithEmailAndPassword
- signInWithEmailAndPassword
- onAuthStateChanged
